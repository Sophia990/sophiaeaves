Group Project · Java · 2024

UniSim

UniSim is a group project I worked on in my second year of university for my Engineering I module. The project involved creating a short 2D game to build a university campus, with the aim to achieve maximum student satifisfaction within time constraints and subject to random events.

My Contributions

My contributions to this project included creating architecture diagrams and creating diagrams and documents to organise the team's time and progress in the first half. In the second half of the project, I was involved in testing, creating a thorough plan for manual testing of the game and writing unit tests for Java.

Through this, I feel that I was able to learn about project management and more about designing the architecture for a game. Mostly, this was the first time I had ensured that a project was thoroughly tested, with both unit tests and the manual test plan that I wrote.
